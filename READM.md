# turnierkaleder-preview

Dieses Repository dient der Veröffentlichung einer Liste von anstehenden Jugendturnieren des Bayerischen Badminton Verbandes. Die Liste erhebt keinen Anspruch auf Korrektheit oder Vollstädigkeit. Sie basiert auf öffentlich verfügbaren Informationen auf https://www.turnier.de bzw. https://badminton-bbv.de.
